Come creare un template per i post

La prima cosa da fare è suddividere in blocchi logici la pagina, identificando le parti comuni a tutte le pagine.

    - Le varie aeree saranno suddivise in:
  • Contenuti di intestazione del tag header
  • Contenuti centrali variabili del tag Main
  • Contenuti di fine pagina comuni nel tag footer

Formattazzione del codice

Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con i css.Percio i passi saranno:

  1. Proggettare il MARKUP assegnando ad ogni specifico tetso il tag appropriato
  2. Scrivere l'HTML dei tag esterni a quelli interni e dall'alto verso il basso
  3. Applicare gli stili css
